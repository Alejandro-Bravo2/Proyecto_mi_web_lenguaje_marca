Aprendiendo Python

¿Qué es Python?

Python es un lenguaje de programación interpretado, fácil de aprender y ampliamente usado. Se utiliza en aplicaciones de inteligencia artificial, desarrollo web, análisis de datos, videojuegos y más. Es conocido por su sintaxis limpia y su enfoque en la legibilidad del código.

Características Principales:

  • Es un lenguaje interpretado: no se compila.
  • Es multiplataforma y compatible con varios sistemas operativos.
  • Soporta programación orientada a objetos, funcional y procedural.
  • Cuenta con una gran comunidad y abundante documentación.

Variables en Python

En Python, las variables se definen de manera dinámica. Ejemplo:

nombre = "Alejandro" # Esto es un string
edad = 25 # Esto es un entero
altura = 1.75 # Esto es un flotante
es_estudiante = True # Esto es un booleano

Conversión de tipos:

Podemos convertir variables de un tipo a otro utilizando funciones como int() # Esto convertiria a tipo entero
str() # Esto convertiria a tipo cadena
float() #Esto convertiria a tipo flotante

Ejemplo:

altura = 1.90 altura_int = int(altura)

Si ahora nosotros le hacemos un print a altura:print(altura_int)Nos devolvería 1

Prueba resolver los siguientes ejercicios para afianzar tus conocimientos:

  1. Crea una variable llamada saludo que contenga la frase "¡Hola Mundo!" y muéstrala por pantalla.

  2. Declara una lista con tus 3 colores favoritos y accede al segundo color.

  3. Escribe un programa que calcule el área de un triángulo dada su base y altura.
