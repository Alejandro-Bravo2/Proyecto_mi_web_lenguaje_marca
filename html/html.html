Aprendiendo Html

¿Qué es Html?

HTML es el lenguaje estándar que se utiliza para estructurar y presentar el contenido en una página web. Es como el esqueleto de una página, definiendo la estructura de la página web . Al aprender HTML, estarás dando el primer paso para crear tus propias páginas web.

Características Principales:

  • Se ejecuta en el navegador
  • Su sintaxis se basa en etiquetas, definiendo el tipo de dato
  • Junto con css permiten una gran personalizan y creación de páginas webs

HEAD

En el head pondremos el titulo, links y lenguaje usado.

Para poner un titulo a nuestra página web:

title = "Alejandro"

Para poner enlaces a hojas de estilos:

link rel="stylesheet" href="../css/principal.css"

Para definir el lenguaje:

meta charset="UTF-8"

BODY

En el body escribiremos la estructura de nuestro código estando esta formada por 3 partes, header, main y footer. En el header escribiremos la información que querramos que aparezca en todas las páginas webs siendo esta reconocida como la cabecera, en el main escribiremos el contenido único de la página web y en footer escribiremos el pie de página añadiendo enlaces como a copy right y otros datos.


Para crear una cabecera:

header

Normalmente en la cabecera tenemos un formulario el cual serán los enlaces a las distintas webs, este podría ser un nav o form

nav>ul>li*4 # Esto sería una estrctura básica

    En el contenido principal escribiremos titulos, parrafos, enlaces, botones o imagenes.

  • p = parrafos
  • h+num = Titulos y sus estilos predefinidos
  • a = enlaces
  • img = imagenes
  • button = botones
  • article y section = para agrupar contenido del mismo tipo
  • input = para preguntar al usuario por datos
  • br = salto linea
