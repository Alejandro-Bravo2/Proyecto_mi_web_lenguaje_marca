Aprendiendo Html

¿Qué es Html?

HTML es el lenguaje estándar que se utiliza para estructurar y presentar el contenido en una página web. Es como el esqueleto de una página, definiendo la estructura de la página web. Al aprender HTML, estarás dando el primer paso para crear tus propias páginas web.

Características Principales:

  • Se ejecuta en el navegador
  • Su sintaxis se basa en etiquetas, definiendo el tipo de dato
  • Junto con css permiten una gran personalizan y creación de páginas webs

HEAD

En el head pondremos el título, links y el conjunto de caracteres usado.

Para poner un título a nuestra página web:

<title>Alejandro</title>

Para poner enlaces a hojas de estilos:

<link rel="stylesheet" href="../css/principal.css" />

Para definir el conjunto de caracteres:

<meta charset="UTF-8" />

BODY

En el body escribiremos la estructura de nuestro código, estando esta formada por 3 partes: header, main y footer. En el header escribiremos la información que queramos que aparezca en todas las páginas web, siendo esta reconocida como la cabecera. En el main escribiremos el contenido único de la página web y en el footer escribiremos el pie de página, añadiendo enlaces como copyright y otros.


Para crear una cabecera:

<header></header>

Normalmente en la cabecera tenemos un formulario o un menú de navegación, el cual podría ser un nav o form.

<nav><ul><li><a href="#">Enlace</a></li></ul></nav>

    Etiquetas comunes en HTML:

  • <p> = párrafos
  • <h+num> = Títulos y sus estilos predefinidos
  • <a> = enlaces
  • <img> = imágenes
  • <button> = botones
  • <article> y <section> = para agrupar contenido del mismo tipo
  • <input> = para preguntar al usuario por datos
  • <br> = salto de línea

Ejercicios Prácticos

Prueba resolver los siguientes ejercicios para afianzar tus conocimientos:

  1. Crea una página HTML que incluya un título y un párrafo describiendo de qué trata.

  2. Añade una imagen y un enlace a tu página HTML.

  3. Crea una lista ordenada y una lista no ordenada con elementos que prefieras.

Soluciones sugeridas al final del documento.
