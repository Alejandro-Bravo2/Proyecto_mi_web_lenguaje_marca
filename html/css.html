Aprendiendo CSS

¿Qué es CSS?

CSS es un lenguaje utilizado para describir el estilo y formato de un documento HTML. Permite agregar color, diseño y estructura a las páginas web.

Características Principales:

  • Separa el contenido de la presentación.
  • Controla el diseño de forma centralizada.
  • Crea experiencias interactivas.

Cómo usar CSS

En CSS tenemos que definir primero a qué elemento queremos hacer referencia; esto lo haremos usando la etiqueta HTML, la clase o el id.

En CSS, los elementos tienen especificidad. Esto significa que algunas reglas tienen mayor prioridad que otras. Es importante entender esto para saber qué estilos se aplicarán a qué elementos.


Para aplicar estilos a un elemento:

.h1 { /* Aquí le daremos sus atributos y luego cerramos con } */ }

    Tipos de datos y propiedades de CSS:

  • color: Define el color del texto. Ejemplo: color: red;
  • background: Establece el color o la imagen de fondo. Ejemplo: background: linear-gradient(blue, white);
  • font: Agrupa atributos relacionados con las fuentes. Ejemplo: font: italic bold 16px Arial;
  • align-items: Alinea los elementos dentro de un contenedor flexbox. Ejemplo: align-items: center;
  • padding: Establece los márgenes internos de un elemento. Ejemplo: padding: 10px;
  • margin: Controla los márgenes externos entre elementos. Ejemplo: margin: 20px auto;
  • display: Define el comportamiento del contenedor. Ejemplo: display: flex;
  • width: Especifica el ancho del elemento. Ejemplo: width: 50%;
  • height: Establece la altura del elemento. Ejemplo: height: 200px;
  • border: Define el borde del elemento. Ejemplo: border: 2px solid black;

Ejercicios Prácticos

Prueba resolver los siguientes ejercicios para afianzar tus conocimientos:

  1. Dale formato a un enlace para que parezca un botón. Haz que, cuando pases el cursor por encima, cambie de color y el texto sea más grande.

  2. Declara una lista con tus 3 colores favoritos y asigna a cada uno un color diferente usando CSS.

  3. AVANZADO: Realiza una cabecera sencilla y aplica estilos para hacer un diseño responsive que se ajuste a diferentes pantallas.
